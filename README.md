## Description
Dowload [Nerd Fonts](https://nerdfonts.com),
Gruvbox cursors - [link](https://github.com/sainnhe/capitaine-cursors),
Gruvbox colorscheme - [link](https://github.com/Fausto-Korpsvart/Gruvbox-GTK-Theme),
Font - JetBrains Mono Nerd Font,
Firefox theme - [link](https://addons.mozilla.org/en-US/firefox/addon/gruvbox-medium-dark/),

## Starship
If [starship](https://github.com/starship/starship) is not installed, follow the official instructions

## Theme
You can use `nwg-look` for apply gruvbox theme for GTK apps

## Editing
Also check out the configs to edit or see what utilities I'm using

## Updates for pacman
Install `pacman-contrib` to see available updates
